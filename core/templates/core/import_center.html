Customer + Sales Import

📋 File Requirements & Format

  • Upload CSV (.csv) or Excel (.xlsx) file.
  • The file must contain both customer and sales data in the same sheet.
  • Each row creates:
    • A Customer (if not already created in file)
    • A Sale linked to that customer
  • Organization is assigned automatically based on your account.
  • Database IDs are auto-generated. Use customer_external_id to link rows internally.

✅ Required Columns

customer_external_id name id_number phone_number country location gender household_type household_size preferred_language registration_date product_type product_name product_model product_serial_number purchase_mode sales_rep type_of_use

🟡 Optional Columns

alternate_phone_number email county sub_county payment_plan referred_by_external_id

📌 Allowed Values

gender: M, F, P, O

(Male, Female, Prefer Not To Say, Other)

household_type: M, F, C, P, O

(Male Headed, Female Headed, Child Headed, Prefer Not To Say, Other)

preferred_language: EN, SW, NA

(English, Swahili, Native Language)

product_type: EPC, IC, O

(Electric Pressure Cooker, Induction Cooker, Other)

purchase_mode: C, DA, P

(Cash, Deposit Account, Paygo)

type_of_use: Domestic, Business, Other

📅 Date Format

Use ISO format: YYYY-MM-DD
Example: 2026-02-06

Transactions Import

📋 File Requirements & Format

  • Upload CSV (.csv) or Excel (.xlsx) file.
  • Each row represents one transaction.
  • Organization is assigned automatically based on your account.
  • txn_id must be unique.
  • No timezone in datetime.
  • transtime must be a 14 digit integer representing East African Time.

✅ Required Columns

time amount txn_id name transtime

🟡 Optional Column

ref

📅 DateTime Format

Use ISO format for column time: YYYY-MM-DD HH:MM:SS
Example: 2026-02-06 12:13:14


Use 24HR format for column transtime: YYMMDDHHmmss
Example: 20260206121314

🧾 Example Row

time amount txn_id name ref transtime
